# 🚀 APIForge

A gamified 4-week API Integration & Automation learning dashboard featuring interactive roadmaps, project-based learning, XP progression, portfolio tracking, internship preparation, freelance readiness, and resource management.

## Features

- 30-Day Learning Path
- Daily Missions & Projects
- XP & Achievement System
- Skill Tree
- Portfolio Tracker
- Internship Tracker
- Freelance Tracker
- Resource Hub
- Obsidian Integration
- Local Storage Persistence
- Responsive SaaS UI

## Technologies

- HTML
- CSS
- JavaScript
- Tailwind CSS
- LocalStorage

## Goal

Help beginners become internship-ready and freelance-ready in API Integration, Automation, and AI Workflows through practical project-based learning.
