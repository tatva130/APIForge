## 🎯 Goal

Help beginners become **internship-ready** and **freelance-ready** in API Integration, Automation, and AI Workflows through practical, project-based learning.

---

## 🚀 Core Features

*   **📅 30-Day Learning Path & Daily Missions:** Step-by-step developer curriculum focusing on real-world projects.
*   **🎮 XP & Achievement System:** Keep learning fun with modular level-ups, tracking streaks, and unlocking milestones.
*   **🗺️ Skill Tree:** A node-based, interactive progression map visualizing technical concepts from foundations to advanced automated systems.
*   **📚 Resource Hub:** A completely curated, searchable, and filterable index of videos, official documentations, and sandbox tools.
*   **💼 Career Hub:** Integrated tracker tooling featuring a **Portfolio Board**, **Internship Tracker**, and **Freelance Contract Dashboard** right in your workspace sidebar.
*   **🧠 Obsidian Integration & Persistence:** Seamless workflows built over `LocalStorage` for high data privacy and retention.
